How it works

This website grabs the information from your ip address, and based on that I input your location into my weather api to call for your area's weather.

The Google Maps Api on the bottom works a bit different. I have to ask if you would like to provide me your exact location, and based on the exact location I have a marker that indicates where you are presently.
